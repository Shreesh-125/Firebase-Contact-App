# Firebase Contact App  

This is a simple contact management application built using **React**, **Firebase**, and **Tailwind CSS**. With this app, users can:  
✅ Add new contacts  
✅ View saved contacts  
✅ Update contact details  
✅ Delete contacts  

All the data is stored in **Firebase Firestore**, ensuring real-time updates and seamless management.  

## **Tech Stack**  
- ⚛️ React.js (Frontend)  
- 🎨 Tailwind CSS (Styling)  
- 🔥 Firebase (Authentication & Firestore)  

---

## **Getting Started**  

Follow these steps to set up the project on your local machine.  

### **1️⃣ Clone the Repository**  
```
git clone https://github.com/YOUR_GITHUB_USERNAME/firebase-contact-app.git
cd firebase-contact-app
```

```
npm install
```

### To start the development server, run:
```
npm run dev
```
